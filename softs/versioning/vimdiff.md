# Vimdiff

### vimdiff

	vimdiff old new

### Raccourcis
```
do - Récupérer les changements de l'autre fenêtre dans la fenêtre courante.
dp - Envoyer les changements de la fenêtre courante vers l'autre fenêtre.
]c - Atteindre le prochain changement.
[c - Atteindre le changement précédent.
Ctrl W + Ctrl W - Basculer d'une fenêtre à l'autre.
```

```
zf : Créer un regroupement
zo : déplier un recgroupement
zc : replier un regroupement

zr : déplier tous les regroupements
zm : replier tous les regroupements
```
