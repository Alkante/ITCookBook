# mkdocs

## Vérification
	python --version
	pip --version

## Install
	sudo apt-get update
	sudo apt-get install python
	sudo apt-get install python-pip

	sudo pip install mkdocs


## Création du projet
### Vérification de version
	mkdocs --version

### Création
	mkdocs new my-project
	cd project

### Run in local
	mkdocs serve

Go to : http://127.0.0.1:8000
