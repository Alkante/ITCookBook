## Install 
### Download nodeJS

[https://nodejs.org/en/](https://nodejs.org/en/)

Extraire le fichier
```bash
tar xJvf node-v6.11.0-linux-x64.tar.xz
rm node-v6.11.0-linux-x64.tar.xz
cd node-v6.11.0-linux-x64
```
